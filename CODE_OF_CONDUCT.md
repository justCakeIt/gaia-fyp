# Code of Conduct  
*Last updated: 08-12-2025*  

**Project Name:** G.a.i.A. – Green AI Alchemy  

This Code of Conduct defines expectations for all contributors and participants in discussions related to this project, including communication on GitHub, pull requests, issue tracking, and any related external interactions.

---

## Expected Behaviour
- Treat all contributors with courtesy and respect.
- Communicate honestly and constructively.
- Provide helpful feedback and consider others' perspectives.
- Acknowledge others’ work and give credit where due.
- Maintain a professional tone and stay on topic.
- Follow project guidelines, branching strategies, and coding practices.

---

## Unacceptable Behaviour
- Harassment, discrimination, or offensive comments.
- Personal attacks or aggressive behaviour.
- Trolling, spamming, or intentionally disrupting discussion.
- Sharing unsafe, misleading, or inappropriate content.
- Ignoring contribution guidelines or licensing rules.

---

## Collaboration Guidelines
- Follow GitHub best practices for issues, branches, and pull requests.
- Commit messages should be clear and meaningful.
- Resolve disagreements respectfully; ask the supervisor for support if needed.
- Ensure all contributions comply with project licensing.
- Protect sensitive data (never commit `.env` or credentials).

---

## Reporting Issues
If you observe or experience a breach of this Code of Conduct, please report it to:  
**stadlerm@roehampton.ac.uk**  
Reports will be treated confidentially.

---

## Consequences
Violations may result in temporary or permanent removal from project spaces depending on severity.
